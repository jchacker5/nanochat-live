## Running the Visualization Script

```bash
# Install dependencies (if not already installed)
pip install numpy matplotlib

# Run the visualization script
python scripts/visualize_srgi_phases.py
```

The script will create all visualizations in the `visualizations/` directory.

## Requirements

- Python 3.10+
- numpy
- matplotlib

Optional (for using actual SRGI components):
- torch
- nanochat modules (ssm, phase_attention, spinor_embeddings, geometric_bottleneck)

The script works with simulated data even if SRGI components are not available.
